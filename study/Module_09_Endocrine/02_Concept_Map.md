 DIABETES emergencies:
   DKA  = Type 1, ketones, HIGH-anion-gap acidosis, glucose ~300-600  (Mod 2)
   HHS  = Type 2, NO ketones, hyperosmolar, glucose VERY high (>600)
   HYPOGLYCEMIA = <70 → treat FAST (15 g carbs awake / D50 or glucagon if not)

 CRISES (can't miss):
   Thyroid storm  = hyper gone wild: fever, tachy, agitation → cool, beta-blocker, antithyroid
   Myxedema coma  = hypo extreme: hypothermia, ↓LOC, bradycardia → thyroid hormone, warm, support
   Adrenal crisis = cortisol collapse: hypotension, hypoglycemia, ↓Na/↑K → IV hydrocortisone + fluids

 RULES: never stop steroids abruptly · SIADH=↓Na (restrict water) · DI=↑Na (replace water/DDAVP)
