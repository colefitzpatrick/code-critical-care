# 07 · NCLEX & Nursing Actions ⭐ (grades + exam)

Transfusion reactions, neutropenic precautions, and sepsis are NCLEX favorites. Do twice.

## A. Priority rule
- **Transfusion reaction → STOP the transfusion FIRST** (then NS, assess, notify). This is THE classic "first action" answer.
- **Sepsis → time-critical Hour-1 bundle.**
- **Neutropenic fever → emergency** (cultures + antibiotics fast).

## B. Transfusion safety (memorize the flow)
1. Verify order, consent, large-bore IV, **only with normal saline** (never LR/dextrose with blood).
2. **Two-RN check** of patient ID + blood product.
3. Baseline vitals → start slowly → **stay first 15 min** (most reactions early) → vitals per policy.
4. Complete within ~4 hours.
5. **Any reaction → STOP, NS to keep vein open, assess, notify provider + blood bank.**

| Reaction | Clues | Note |
|---|---|---|
| Febrile | fever/chills | most common |
| Allergic | hives → anaphylaxis | antihistamine; epi if severe |
| **Acute hemolytic** | fever, flank/back pain, dark urine, shock | **ABO mismatch — deadly** |
| TACO | dyspnea, HTN, crackles, JVD | overload → slow/stop, diuretic |
| TRALI | acute hypoxia, no overload | lung injury — respiratory support |

## C. Drugs → why → watch
| Drug | Use | Watch / reversal |
|---|---|---|
| Heparin | anticoagulation | aPTT; **HIT** (drop in platelets); reverse = **protamine** |
| Warfarin | anticoagulation | INR; reverse = **vitamin K / FFP** |
| DOACs | anticoagulation | no routine labs; specific reversal agents |
| Iron / B12 / folate | anemia | type-specific; B12 for pernicious anemia |
| Filgrastim (G-CSF) | boost neutrophils | bone pain |
| Norepinephrine | septic shock | MAP ≥65 |
| Broad-spectrum antibiotics | sepsis/neutropenic fever | **cultures BEFORE first dose** |

## D. Labs you act on
- **Hgb/Hct** (anemia/bleed) · **WBC + ANC** (infection/neutropenia) · **platelets**
- **PT/INR** (warfarin) · **aPTT** (heparin) · **fibrinogen + D-dimer** (DIC) · **lactate** (sepsis)

## E. Teaching
- **Anticoagulants:** bleeding precautions, no NSAIDs without asking, consistent vitamin K (warfarin), follow-up labs, medical-alert ID.
- **Neutropenia:** avoid crowds/sick contacts, hand hygiene, no raw/undercooked foods or fresh flowers (per policy), report **fever immediately**.
- **Anemia:** iron with vitamin C (absorption), expect dark stools with iron, B12 for pernicious anemia.

## F. Safety & delegation
- **RN only:** hang/verify blood, manage reactions, sepsis/DIC management, teaching, first assessment.
- **UAP can:** vitals (report promptly during transfusion), ADLs, hygiene — RN interprets.
- **Neutropenic precautions** and **transfusion verification** are RN-led safety processes.

## G. NCLEX test-taking
- Transfusion reaction = **STOP the transfusion** (before notifying, before vitals interpretation).
- Blood is given **only with normal saline.**
- DIC → **treat the underlying cause**; expect ↓platelets + ↑PT/aPTT + ↑D-dimer.
- Neutropenic fever = emergency → cultures + antibiotics; don't wait.
- Sepsis → Hour-1 bundle; cultures **before** antibiotics; norepinephrine to MAP ≥65.
- Heparin + dropping platelets = think **HIT** (stop heparin).
