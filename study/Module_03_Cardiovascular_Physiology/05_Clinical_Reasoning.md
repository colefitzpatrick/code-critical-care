# 05 · Clinical Reasoning

Say it before reading. **Cue → which term moved → effect on MAP → nurse action.**

### Case 1 — Trauma, BP 84/60, HR 130, cool skin
- **Reasoning:** blood loss → ↓preload → ↓SV → ↓CO → ↓MAP; baroreflex → ↑HR, vasoconstriction (cool skin).
- **Action:** control bleeding, isotonic fluids/blood (refill preload), trend BP/HR/lactate, escalate. Hypovolemic shock = empty tank.

### Case 2 — Sepsis, BP 80/45, HR 118, warm/flushed, lactate 5
- **Reasoning:** vasodilation → SVR↓ → MAP falls directly; leak → ↓preload too. "Warm shock."
- **Action:** Hour-1 bundle — fluids (preload) + **norepinephrine** (SVR) to **MAP ≥ 65**; antibiotics + source control. (Now you know the *why* from the equations.)

### Case 3 — HR 175 (SVT), BP 78/40, dizzy
- **Reasoning:** diastole too short → poor filling → ↓SV → CO falls despite huge rate.
- **Action:** unstable tachyarrhythmia → rate control / synchronized cardioversion per protocol; don't wait. Fast ≠ good output.

### Case 4 — Big anterior MI, BP 82/58, crackles, cool
- **Reasoning:** dead muscle → ↓contractility → ↓SV → ↓CO; backs up into lungs (crackles). Cardiogenic shock = broken pump.
- **Action:** this is NOT a fluid-bolus problem (pump, not tank) — support contractility/perfusion, reperfuse the artery, careful monitoring. Differentiating the broken term changes the treatment.

### Case 5 — Post-op, BP 100/78 (narrow), HR 122, "fine"
- **Reasoning:** narrow pulse pressure + tachycardia = baroreflex compensating for early volume loss; MAP still propped up.
- **Action:** don't be fooled by "normal-ish" BP — investigate (bleeding?), trend, escalate early. Compensated ≠ stable.

## The transferable move
Cue → is it **preload, contractility, afterload, SVR, or rate**? → predict MAP →
treat the broken term. Same engine as Modules 01–02, now on hemodynamics.
