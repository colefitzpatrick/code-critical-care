## TIER 1 — full map (copy once, out loud)
```
        VENOUS RETURN
             │ (more in → more stretch)
             ▼
        PRELOAD ───┐
                    │  Frank–Starling
        CONTRACTILITY ──► STROKE VOLUME (SV) ◄── AFTERLOAD (opposes ejection)
                    │
                    ▼
              CO = HR × SV
                    │
                    ▼
            MAP = CO × SVR ──────────────► ORGAN PERFUSION
                    ▲                          │
                    │                          ├─ heart muscle: in DIASTOLE
            BAROREFLEX (neg. feedback)         └─ brain/kidney (urine = gauge)
            low MAP → ↑HR, ↑SVR, ↑contractility

  LEVERS (what moves MAP):
    ↓ preload (bleed, dehydration)      → ↓SV → ↓CO → ↓MAP
    ↓ contractility (MI, HF)            → ↓SV → ↓CO → ↓MAP
    ↑ afterload (vasoconstriction/HTN)  → harder ejection (↓SV if pump weak)
    ↓ SVR (sepsis vasodilation)         → ↓MAP directly  (Module 01 link)
    HR too fast                          → ↓filling time → ↓SV → CO can FALL
```
