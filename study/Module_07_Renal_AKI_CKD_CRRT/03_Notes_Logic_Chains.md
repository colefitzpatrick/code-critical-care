# 03 · Notes as Logic Chains

*Because X → therefore Y → so the nurse does Z.* Bold → `_GLOSSARY.md`.

## Chain 1 — GFR is flow-driven (the shock connection)
Filtration is pressure-driven → **GFR depends on renal blood flow / MAP**
**→** ↓MAP (shock, hypovolemia, HF) → ↓GFR → ↓urine = **pre-renal AKI** (most common)
**→** if low flow persists → tubular cells run out of ATP (Mod 01 Chain 1) → they die =
**ATN (acute tubular necrosis)** = pre-renal slides into **intrinsic** AKI
**→ nurse:** **urine output ≥0.5 mL/kg/hr** is your continuous perfusion gauge; oliguria =
treat the cause (fluids if hypovolemic, perfusion if shock). Catch pre-renal early before
it becomes ATN. **BUN:Cr >20:1** flags pre-renal (kidney still avidly reabsorbing).

## Chain 2 — The 3 AKI types (classify every one)
**Pre-renal** = flow problem (fix perfusion) · **Intrinsic** = tissue damage (ATN /
nephrotoxins — supportive, time) · **Post-renal** = obstruction (relieve it — catheter,
stent, stones)
**→ nurse:** the type changes the action — fluids help pre-renal but *flood* an intrinsic/
overloaded kidney; a blocked catheter (post-renal) is fixed by unblocking, not fluids.
Always ask "flow in, tissue, or flow out?"

## Chain 3 — Nephrotoxins (preventable intrinsic AKI)
Certain drugs/agents poison tubular cells: **contrast dye, NSAIDs, aminoglycosides
(gentamicin), vancomycin, ACE inhibitors** (in low-flow states)
**→ nurse:** check renal function before contrast/nephrotoxins; **hydrate around contrast**;
monitor Cr/levels (vanco/gent troughs); avoid NSAIDs in at-risk patients; hold ACE if AKI.

## Chain 4 — Failed kidney → the lethal trio (AEIOU)
Kidney can't excrete K⁺, can't make HCO₃⁻, can't dump water
**→** **hyperkalemia** (the killer — Mod 1 Chain 4) + **metabolic acidosis** (Mod 2; which
*worsens* hyperK via H⁺/K⁺ swap) + **fluid overload** (→ pulmonary edema, HTN) + **uremia**
**→ nurse:** monitor **K⁺ + ECG**, treat hyperK (stabilize/shift/remove), restrict fluid +
Na⁺ + K⁺ + phosphorus, daily weight & strict I&O, and recognize when **dialysis** is needed
(AEIOU: Acidosis, Electrolytes, Intoxication, Overload, Uremia).

## Chain 5 — AKI has phases (each has its own danger)
**Onset → Oliguric** (low urine: fluid overload, ↑K⁺, acidosis, uremia — *most dangerous*)
**→ Diuretic** (kidney recovers but dumps huge urine: now risk **hypovolemia + hypokalemia**)
**→ Recovery**
**→ nurse:** in oliguric phase watch overload + hyperK; in **diuretic phase flip your
worry** — replace fluids and watch for **LOW** K⁺. The danger reverses between phases.

## Chain 6 — CKD = slow nephron loss → "A WET BED" all fail chronically
Progressive loss (diabetes #1, HTN #2) → staged by **GFR** (Stage 5/ESRD <15 → dialysis)
**→** chronic consequences: **anemia** (↓EPO), **bone–mineral disease** (↓vit D → ↓Ca →
↑PTH + ↑phosphate → bone breakdown), **hyperkalemia**, **metabolic acidosis**, **fluid
overload/HTN**, **uremia**
**→ nurse:** phosphate binders with meals, EPO + iron for anemia, vit D/Ca management,
fluid/K⁺/phos/protein diet limits, BP control, prep for dialysis access; teach adherence.

## Chain 7 — Dialysis & access
**Hemodialysis (HD)** = fast intermittent (3–4 h, 3×/wk) — needs stable BP ·
**CRRT** = slow continuous 24/7 for **hemodynamically unstable** ICU patients (Mod 2) ·
**Peritoneal** = uses the peritoneum, done at home
**→ Access (AV fistula/graft):** the lifeline — **protect that arm: NO BP cuff, NO IV, NO
venipuncture in it.** Assess patency: feel the **thrill** (buzz) + hear the **bruit**
(whoosh). No thrill/bruit = clotted = emergency.
**→ nurse:** between HD sessions K⁺ climbs (the missed-dialysis hyperK from Mod 1); time
nephrotoxic/renally-cleared meds around dialysis; watch BP shifts during/after HD.

---
### Compression (20 s)
> Kidney = A WET BED, all flow-driven. AKI is pre-renal (flow), intrinsic (ATN/toxins),
> or post-renal (obstruction). Failure → hyperK + acidosis + overload + uremia (AEIOU →
> dialysis). Oliguric phase = overload/↑K; diuretic phase = ↓volume/↓K. Protect the
> fistula arm.
