## TIER 1 — full map (copy once, by hand, out loud)

```
                  ┌─────────────────────────────────────────┐
                  │  BODY = buffered salt-water tank, pH-     │
                  │  and volume-regulated                     │
                  └───────────────┬───────────────────────────┘
            ┌─────────────────────┼─────────────────────────┐
            ▼                     ▼                          ▼
   ┌─────────────────┐  ┌──────────────────┐   ┌────────────────────────┐
   │ VOLUME           │  │ ELECTROLYTES     │   │ ACID–BASE              │
   │ water follows Na⁺│  │ Na⁺ = water/osm  │   │ buffer: CO₂+H₂O ⇌      │
   │ hypo↔hypervolemia│  │ K⁺ = cardiac     │   │   H₂CO₃ ⇌ H⁺ + HCO₃⁻   │
   └────────┬─────────┘  │ Ca²⁺/Mg²⁺ =      │   └───────────┬────────────┘
            │            │   nerve/muscle   │               │
            │            └──────────────────┘     ┌─────────┴──────────┐
            │                                      ▼                    ▼
            │                            LUNGS (fast)            KIDNEYS (slow)
            │                            change PaCO₂            change HCO₃⁻
            │                            = the ACID term         = the BASE term
            ▼                                      │                    │
   assess: BP, HR, urine,                          └─────► pH = HCO₃⁻ / PaCO₂
   skin, weight, edema                                     (base / acid ratio)

   4 PRIMARY ACID–BASE DISORDERS (ROME: Resp Opposite, Metab Equal)
     Respiratory acidosis  : ↑PaCO₂  → ↓pH   (hypoventilation)
     Respiratory alkalosis : ↓PaCO₂  → ↑pH   (hyperventilation)
     Metabolic acidosis    : ↓HCO₃⁻  → ↓pH   (lactate, DKA, diarrhea, renal)
     Metabolic alkalosis   : ↑HCO₃⁻  → ↑pH   (vomiting, diuretics)
       → the OTHER controller COMPENSATES (toward, never past, normal)

   COUPLING (link to Module 01):
     acidosis ↔ hyperkalemia (H⁺ in, K⁺ out)   |   vomiting → alkalosis + ↓K⁺
     shock → lactic acidosis (Mod 01 Ch1/Case5) |  Na⁺ problem = usually a WATER problem
```
