# 05 · Clinical Reasoning

**Cue → mechanism → priority action.** Say it before reading.

### Case 1 — HF patient: +4 lb overnight, dyspnea, crackles, O₂ 89%
- **Mechanism:** decompensated left HF → pulmonary congestion.
- **Priority:** ABCs first — **sit upright (high-Fowler's), O₂**, notify provider,
  anticipate IV diuretic, daily weight/I&O, continuous monitor. (Airway/oxygenation > all.)

### Case 2 — Crushing chest pain, diaphoretic, nausea
- **Mechanism:** likely ACS/MI — coronary occlusion.
- **Priority:** **ECG ≤10 min**, aspirin, O₂ if <90%, nitro, pain, IV/monitor, alert
  provider/cath. Time = muscle. Don't delay for labs.

### Case 3 — HF patient on furosemide: K⁺ 2.9, leg cramps, on digoxin
- **Mechanism:** loop diuretic → hypokalemia; low K⁺ + **digoxin** → toxicity risk
  (Mod 01 Ch4 — competitive binding).
- **Priority:** hold/clarify, monitor ECG, replace K⁺ (never push), check digoxin level,
  notify. *Three modules converge here.*

### Case 4 — New AF, irregular pulse, slurred speech 1 hr ago
- **Mechanism:** AF → atrial stasis → clot → embolic **stroke**.
- **Priority:** this is a stroke — rapid neuro/stroke protocol activation, monitor,
  anticipate imaging; longer-term anticoagulation for the AF.

### Case 5 — Big MI, now BP 78/50, cool, crackles
- **Mechanism:** cardiogenic shock (broken pump).
- **Priority:** NOT a fluid bolus (floods the failing pump) — support perfusion/
  contractility, urgent reperfusion, ICU. Identify "pump, not tank."

## Transferable move
Cue → which problem (pump/pipe/electric)? → which compensation or term broke? →
ABC-priority action. Same engine, cardiac domain.
