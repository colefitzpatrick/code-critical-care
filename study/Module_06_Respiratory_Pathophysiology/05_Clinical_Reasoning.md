# 05 · Clinical Reasoning

**Cue → which failure mode → priority action (A/B first).**

### Case 1 — Asthma flare in ED, RR 36, wheezing now silent
- **Mechanism:** severe bronchospasm → **silent chest** (no air moving) → near arrest.
- **Priority:** **emergency.** O₂, sit upright, continuous neb albuterol + ipratropium, IV steroids, IV magnesium, prepare for intubation now — don't wait. Call code/RT.

### Case 2 — COPD patient, sleepy on 6 L NC, SpO₂ 96%
- **Mechanism:** likely over-oxygenated → CO₂ retention worsening → drowsy (CO₂ narcosis).
- **Priority:** **titrate O₂ down** to ordered SpO₂ target (~88–92%), get an ABG, monitor mental status; do NOT just yank O₂ off — wean. Anticipate BiPAP if CO₂ keeps climbing.

### Case 3 — Septic patient, SpO₂ 84% on 100% NRB, bilateral infiltrates
- **Mechanism:** **ARDS** (sepsis is the #1 cause) → widespread shunt → refractory hypoxemia.
- **Priority:** escalate now — anticipate intubation + lung-protective vent (Vt 6 mL/kg IBW, PEEP), possible prone, treat sepsis (Hour-1 bundle); more FiO₂ alone won't save them.

### Case 4 — Post-op day 2 hip replacement, sudden dyspnea, SpO₂ 86%, clear lungs, HR 122
- **Mechanism:** **PE** (Virchow's: stasis + hypercoag + injury) → dead space.
- **Priority:** O₂, sit upright, IV/monitor, alert provider, anticipate CTPA + therapeutic anticoagulation; if unstable → thrombolytics pathway.

### Case 5 — Vented ARDS patient, peak pressures climbing, SpO₂ falling
- **Mechanism:** possible mucus plug, tube migration, pneumothorax, or worsening ARDS.
- **Priority:** **DOPE** check — **D**isplaced tube, **O**bstruction (suction), **P**neumothorax (auscultate, CXR), **E**quipment (vent/disconnect). Bag-valve if needed; call respiratory + provider.
