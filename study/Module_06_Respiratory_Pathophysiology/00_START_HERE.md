# Module 06 — Respiratory Pathophysiology + Vent Intro · START HERE 🔓

⏱️ ~6 sessions. Science + **amplified nursing** (`07_NCLEX_and_Nursing_Actions.md`).
Builds directly on M05 (lungs that work) → M06 (lungs that fail), with the bridge to ICU mechanical ventilation.

## Why
M05 gave you ventilation, oxygenation, V/Q, and the failure types. M06 is the four ways the lung *breaks* — obstruction, shunt, dead space, drive — and what the nurse does. Plus your first real ventilator vocabulary. This is one of the highest-yield ICU/CCRN blocks.

## What "mastered" looks like
You can name the failure mode behind any common diagnosis (ARDS / COPD / asthma / PE / pneumonia / atelectasis), predict its ABG, and state the first 3 nurse actions cold.
