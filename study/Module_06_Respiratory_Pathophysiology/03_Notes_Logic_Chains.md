# 03 · Notes as Logic Chains

*Because X → therefore Y → so the nurse does Z.* Bold → `_GLOSSARY.md`.

## Chain 1 — Asthma (obstruction + inflammation + mucus)
Trigger (allergen/cold/exercise) → bronchospasm + airway inflammation + mucus plugging
**→ Type 2 picture if severe:** trapped air, CO₂↑, pH↓ (resp acidosis), hypoxemia
**→ Status asthmaticus warning sign: SILENT chest** (no air moving) = near arrest
**→ nurse:** sit up, O₂ to target, **rescue bronchodilator (albuterol) THEN steroid**, IV steroid in severe, anticipate continuous nebs, **Mg sulfate IV**, prepare for intubation if exhausted/silent chest.

## Chain 2 — COPD (emphysema + chronic bronchitis)
Chronic destruction of alveoli (emphysema) + chronic airway inflammation (bronchitis) → air trapping → **chronic CO₂ retention** (compensated resp acidosis from Mod 02)
**→ Exacerbation:** infection/pollutant → worse obstruction → **Type 2 failure**
**→ nurse:** **titrate O₂ to ordered target (~88–92%)** — *don't withhold* but don't blast (over-oxygenation can worsen CO₂); bronchodilator + steroid + ABx if infection; consider **BiPAP** before intubation in COPD exacerbation (huge mortality benefit); teach **pursed-lip breathing**, smoking cessation, vaccines.

## Chain 3 — ARDS (diffuse alveolar damage → SHUNT)
Severe insult (sepsis #1, pneumonia, aspiration, trauma) → cytokine storm → alveolar membranes leaky → alveoli flood → **widespread shunt** → hypoxemia that **does not** improve with more FiO₂
**→ Berlin criteria (simplified):** acute, bilateral infiltrates, not cardiac, low PaO₂/FiO₂
**→ nurse:** anticipate **lung-protective ventilation** (Vt **6 mL/kg ideal body weight** + PEEP), **prone positioning** for severe (improves V/Q + recruits posterior alveoli), permissive hypercapnia accepted, sedation/paralysis as needed, treat underlying cause (sepsis Hour-1!).

## Chain 4 — Pulmonary embolism (PE → DEAD SPACE)
Clot (usually from DVT, Virchow's triad: stasis + hypercoagulability + endothelial injury) → lodges in pulmonary artery → **alveoli ventilated but not perfused** → wasted breaths
**→** sudden dyspnea + sharp pleuritic chest pain + tachycardia + hypoxia + clear lungs
**→ diagnosis pathway:** Wells score → D-dimer if low-risk → **CT pulmonary angiography** in stable patient
**→ nurse:** O₂, sit upright, IV access, monitor, anticipate **anticoagulation** (heparin → DOAC/warfarin), **thrombolytics** if massive/unstable; prevent: early ambulation + SCDs + chemo-prophylaxis in hospital.

## Chain 5 — Pneumonia (fill alveoli → shunt + impaired diffusion)
Pathogen → alveoli fill with pus/inflammatory fluid → **shunt** + **thicker membrane** (Fick) → hypoxemia + work of breathing↑
**→ signs:** fever, productive cough, crackles, ↓SpO₂, focal infiltrate on CXR
**→ severity:** CURB-65 (Confusion, Urea↑, RR≥30, BP low, age ≥65)
**→ nurse:** O₂ to target, **cultures THEN antibiotics** (sepsis bundle if septic), positioning (good lung down if unilateral, but high-Fowler's for ease of breathing), incentive spirometer/cough, hydration, monitor for sepsis.

## Chain 6 — Pneumothorax / chest tubes
Air enters pleural space → lung partially or completely collapses
**→ Simple pneumothorax:** dyspnea, ↓breath sounds on side
**→ TENSION pneumothorax (emergency):** **tracheal deviation AWAY from side**, absent breath sounds, hypotension, JVD → **immediate needle decompression**, then chest tube
**→ Chest tube water-seal chamber:** tidaling = normal, continuous bubbling = air leak (assess + report), sudden cessation with worsening = possible obstruction → emergent
**→ nurse:** keep system below chest, never clamp routinely, monitor drainage, occlusive dressing if dislodged, daily CXR.

## Chain 7 — Mechanical ventilation basics
3 main knobs: **FiO₂** (oxygenation), **PEEP** (keeps alveoli open), **Tidal volume Vt** (ventilation/CO₂)
**→ Lung-protective in ARDS:** Vt 6 mL/kg IBW + PEEP titrated to oxygenation + permissive hypercapnia (accept higher PaCO₂ to avoid barotrauma)
**→ Vent weaning readiness:** improving cause, awake/cooperative, adequate gas exchange on low FiO₂ (≤40%) and low PEEP (≤5), hemodynamically stable → **spontaneous breathing trial**
**→ nurse priorities:** HOB ≥30° (VAP prevention), oral care, sedation interruption daily, DVT/ulcer prophylaxis, monitor airway/cuff/sounds, suction PRN with hyperoxygenation (≤10–15 sec/pass, watch SpO₂/HR), recognize deterioration (rising peak pressures, falling sats).

---
### Compression (20 s)
> Lung fails 4 ways — obstruction (asthma/COPD), shunt (ARDS/pneumonia), dead space (PE), drive (opioids). ABG tells you which. Shunt doesn't fix with O₂ alone — needs PEEP. PE is dead space — anticoag. Vent has 3 knobs: FiO₂, PEEP, Vt; ARDS = lung-protective 6 mL/kg + PEEP.
