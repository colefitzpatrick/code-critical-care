## How this system works (read once)

Every topic is a **Module = one folder**. You do the folders **in order**. Inside every
module the files are **numbered in the exact order you work them**:

| File | What it does | Your move |
|------|--------------|-----------|
| `00_START_HERE` | The goal, the "why it matters for nursing," the order, the checklist | Read first, 3 min |
| `01_Big_Picture` | Broad orientation + the core logic chain. **Start broad.** | Read + understand the "why" |
| `02_Concept_Map` | The map you redraw from memory (3 difficulty tiers) | Draw it. Again. And again. |
| `03_Notes_Logic_Chains` | Notes written as cause → effect → consequence | Read to *reason*, not memorize |
| `04_Recall_Sets` | Small recall sets (5–8 Qs each) with checkboxes | Active recall, blank page |
| `05_Clinical_Reasoning` | "Where you'll see this as a nurse" | Train the pattern, not the fact |
| `06_Spaced_Review` | Day 3 / 10 / 24 cumulative mini-quizzes | Lock it to permanent memory |
| `_GLOSSARY` | Every medical term in the module, plain English + engineering link | Look up as needed (not front-to-back) |
| `_MISSES_TO_SEND` | Your gap log → hand back to me to regenerate harder material | Add a line whenever you blank |

> **Vocabulary convention:** you do not have medical vocabulary yet — that is expected
> and planned for. Medical terms are **bolded on first use** and defined in plain English
> (with an engineering link where it helps) in that module's `_GLOSSARY.md`. Never push
> past a term you don't know — it's one lookup away. Future modules define terms inline,
> every time.

**The loop:** read broad → draw the map → reason through notes → recall in small sets →
apply clinically → space it. You only ever face a small set, never a wall of content.

## 🎯 The dopamine rules (non-negotiable, ADHD-proofed)

- **One set is a win.** The minimum unit is *one* recall set (~5–8 questions, ~10 min).
  Doing one set = a checked box = done for the day if that's all you've got.
- **Never zero.** A bad day = the 5-question mini-set in `06_Spaced_Review`, not nothing.
- **Check the box the moment you finish** — the box is the reward, protect it.
- **Stop at the set boundary.** Momentum is good; overrun-until-burnout is not.
- **Misses are fuel, not failure.** Every wrong recall answer → drop it in
  `_MISSES_TO_SEND.md`. You hand me that list; I rebuild it harder and sharper for you.

later links back to Module 01.

  > feedback loop broke." This is the anchor you link everything to.

## ✅ Mastery bar (when is a module "done")

A module is done — and the next unlocks — when **all of these are true**:

- [ ] You can redraw `02_Concept_Map` Tier 3 (blank page, from memory) correctly
- [ ] You can answer every set in `04_Recall_Sets` cold, explaining the **why**
- [ ] You can talk through every scenario in `05_Clinical_Reasoning` out loud
- [ ] You passed the Day 24 quiz in `06_Spaced_Review`

That's the real bar — *explain it, draw it, use it* — because that's what makes you a
good nurse, and the test then takes care of itself.
