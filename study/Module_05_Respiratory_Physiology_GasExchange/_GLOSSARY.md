# _GLOSSARY — Module 05 (plain English)

Lookup. **term** — meaning — *(link)*.

- **Ventilation** — moving air in/out; controls **CO₂**. (Failure → ↑CO₂, Type 2.)
- **Oxygenation** — loading **O₂** into blood; measured by SpO₂/PaO₂. (Failure → Type 1.)
- **V/Q matching** — matching ventilation (air) to perfusion (blood) for efficient exchange.
- **Shunt** — alveoli perfused but **not ventilated** (flooded/collapsed) → hypoxemia O₂ barely fixes. *(ARDS, pneumonia.)*
- **Dead space** — alveoli ventilated but **not perfused** → wasted breaths. *(PE, low CO.)*
- **Diffusion** — gas crossing the alveolar membrane; Fick: ∝ area × ΔP ÷ thickness.
- **PaO₂ / PaCO₂** — arterial O₂ / CO₂ pressure (from an ABG).
- **SpO₂** — % of hemoglobin saturated with O₂ (a pulse-ox estimate; not O₂ *delivery*).
- **Hemoglobin (Hb)** — carries most O₂ in blood.
- **O₂–Hb dissociation curve** — saturation vs O₂; **right shift** (fever, acid, ↑CO₂, 2,3-BPG) = easier tissue unloading.
- **Type 1 respiratory failure** — oxygenation failure (low O₂, CO₂ normal/low).
- **Type 2 respiratory failure** — ventilation failure (high CO₂ → respiratory acidosis, Mod 02).
- **ARDS** — diffuse alveolar flooding → massive shunt → refractory hypoxemia.
- **COPD / asthma** — airflow obstruction; air trapping, CO₂-retention tendency.
- **PE (pulmonary embolism)** — clot blocks pulmonary blood flow → dead space + sudden dyspnea.
- **Atelectasis** — collapsed alveoli (post-op) → shunt; prevented by deep breathing/incentive spirometry.
- **PEEP** — ventilator pressure that holds alveoli open (recruits shunted lung).
- **Pursed-lip breathing** — slow exhale technique that keeps airways from collapsing (COPD).
- **Incentive spirometer** — device to encourage deep breaths, prevent atelectasis.
- **Capnography / EtCO₂** — exhaled-CO₂ monitor; catches hypoventilation early (Mod 01 link).
