```
        DRIVE (↑CO₂/↓pH → central chemoreceptors)
                       │
                       ▼
              VENTILATION (move air)  ──controls──► CO₂  (PaCO₂)
                       │
                       ▼  must match
        ┌──────────  V / Q  MATCHING  ──────────┐
        ▼                                        ▼
   SHUNT (perfused,                        DEAD SPACE (ventilated,
   not ventilated)                          not perfused)
   → hypoxemia NOT fixed                    → wasted ventilation,
     by more O₂  (ARDS, pneumonia)            ↑CO₂ (PE, low CO)
        │                                        │
        └──────────► DIFFUSION at alveolus ◄─────┘
                       │ (Fick: area, ΔP, thickness)
                       ▼
              O₂ onto HEMOGLOBIN → tissues
              O₂–Hb curve: RIGHT shift (fever, acid, ↑CO₂,
              ↑2,3-BPG) = unload O₂ to tissues more easily

   FAILURE TYPES:  Type 1 = oxygenation (↓O₂, normal/low CO₂)
                   Type 2 = ventilation (↑CO₂ → resp acidosis, Mod 02)
   PRIORITY: Airway → Breathing → everything else
```
