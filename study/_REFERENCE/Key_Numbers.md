# Quick Reference — Numbers to Know Cold

> Memorize these. They're what every ABSN exam, NCLEX item, and ICU report sheet runs on.
> Fishbones below are the **lab shorthand** used on whiteboards and in SBAR everywhere.

---

## 🫀 Vital signs — adult

| Sign | Normal | "Act on" thresholds |
|---|---|---|
| HR | 60–100 | <50 or >120 sustained |
| RR | 12–20 | <10 or >24 sustained |
| BP | <120/80 | SBP <90 or >180; SBP <90 = shock workup |
| MAP | ≥65 (ICU target) | <65 = perfusion failing |
| Temp | 36.5–37.5 °C (97.7–99.5 °F) | <36 or ≥38 = SIRS criterion |
| SpO₂ | ≥95% (88–92% target in COPD) | <90% — act |
| Pain | 0–10 scale | Treat + reassess |

## 🩸 ABG — normals + the shorthand

| pH | PaCO₂ | HCO₃⁻ | PaO₂ | SaO₂ | Anion gap |
|---|---|---|---|---|---|
| **7.35–7.45** | **35–45** | **22–26** | **80–100** | **≥95%** | **8–12** |

- **Acidemia** <7.35 · **Alkalemia** >7.45
- **ROME** — Resp Opposite (pH↔CO₂), Metab Equal (pH↔HCO₃⁻)
- **Winter's formula** (metab acidosis): expected PaCO₂ ≈ **1.5 × HCO₃⁻ + 8 ± 2**
- **Anion gap** = Na⁺ − (Cl⁻ + HCO₃⁻). High → added acid (lactate/DKA/tox). Normal → lost HCO₃⁻ (diarrhea/RTA).

## 🦴 BMP / Chem-7 fishbone (the classic skeleton)

```
              Na⁺    │   Cl⁻    │   BUN     \
              135-145│  98-107  │   7-20     \
              ───────┼──────────┼──────────  ── Glucose 70–100
              K⁺     │  HCO₃⁻   │   Creat   /     (fasting)
              3.5-5.0│  22-26   │  0.6-1.2 /
```

Reading order on the whiteboard: top-left = Na⁺, bottom-left = K⁺, middle Cl⁻/HCO₃⁻, right BUN/Cr, isolated point = glucose.

## 🩸 CBC fishbone

```
                          WBC
                         4.5-11
                    ──────┼──────
       Hgb (M 13.5-17.5 / F 12-15.5)     Hct (M 41-53% / F 36-46%)
                    ──────┼──────
                         Plt
                      150-400 K
```

Or written linearly: `WBC | Hgb / Hct | Plt`. Low Hgb/Hct = anemia or bleed; low Plt = bleeding risk; WBC up = infection, down = immune compromise.

## 🩹 Coags

| PT | INR | aPTT |
|---|---|---|
| 11–13.5 sec | 0.8–1.1 (warfarin target **2–3**) | 25–35 sec |

- **Warfarin** → INR · **Heparin** → aPTT (often 1.5–2.5× normal)
- DOACs (apixaban/rivaroxaban) → no routine monitoring

## 💔 Cardiac / HF / shock

| Marker | Normal | Notes |
|---|---|---|
| Troponin | <0.04 ng/mL (lab-specific) | ↑ = myocardial injury / MI |
| BNP | <100 pg/mL | >400 strongly suggests HF |
| Lactate | <2 mmol/L | 2–4 watch · ≥4 severe / sepsis |
| EF | 55–70% | <40% = HFrEF (systolic failure) |
| CO | 4–8 L/min | CO = HR × SV |
| CVP | 2–8 mmHg | Right-sided preload |
| PAWP (wedge) | 6–12 mmHg | Left-sided preload |
| MAP | ≥65 target | MAP = CO × SVR |

## ⚡ Electrolytes — full set

| Lyte | Normal | Critical |
|---|---|---|
| **Na⁺** | 135–145 mEq/L | <125 or >155 → neuro precautions |
| **K⁺** | 3.5–5.0 mEq/L | <2.5 or >6.0 → ECG + call |
| **Cl⁻** | 98–107 | — |
| **HCO₃⁻** | 22–26 | — |
| **Ca²⁺ (total)** | 8.5–10.5 mg/dL | <7 → tetany/airway · >12 → coma |
| **iCa²⁺** | 1.1–1.3 mmol/L | The biologically active form |
| **Mg²⁺** | 1.7–2.2 mg/dL | Low → refractory K⁺/torsades — *always check with K* |
| **Phos** | 2.5–4.5 mg/dL | Low → refeeding/DKA recovery |

## 🩺 Renal / Urine

- BUN 7–20 · Cr 0.6–1.2 · BUN:Cr ratio ~10–20:1 (>20:1 = pre-renal/hypovolemic)
- **Urine output ≥ 0.5 mL/kg/hr** · oliguria < 0.5 · anuria essentially none
- AEIOU for dialysis/CRRT: **A**cidosis · **E**lectrolytes (hyperK) · **I**ntoxications · **O**verload · **U**remia

## 🍬 Glucose / A1c

| | Value |
|---|---|
| Fasting | 70–100 |
| Random | <140 |
| A1c — normal | <5.7% |
| A1c — prediabetic | 5.7–6.4% |
| A1c — diabetic | ≥6.5% |
| Hypoglycemia | <70 — treat |
| DKA hallmark | Glucose >250 + ketones + high-AG metabolic acidosis |

## 🧠 Neuro

- **GCS** 3 (worst) → 15 (normal); **≤8 = consider intubation** ("less than 8, intubate")
- **ICP** 7–15 mmHg · >20 = escalate · brain perfusion pressure (CPP) = MAP − ICP
- Pupils: PERRLA (Pupils Equal, Round, Reactive to Light, Accommodation)

## 🫁 Respiratory

- RR 12–20 · TV ~500 mL · SpO₂ ≥95% (COPD target 88–92%)
- **Type 1 failure** = low O₂ (oxygenation); **Type 2** = high CO₂ (ventilation → resp acidosis)
- Mechanical vent quick set: tidal volume 6 mL/kg ideal body weight (lung-protective in ARDS); PEEP starts ~5

## 💊 Drug-related quick numbers

- **Digoxin**: hold if HR <60 (apical 1 min); narrow therapeutic; ↓K⁺ ↑toxicity; yellow halos = tox
- **Beta-blocker**: hold if HR <60 or SBP low per protocol; don't stop abruptly
- **Nitroglycerin**: 1 SL q5 min × 3; call 911 if no relief; *never* with sildenafil/PDE-5
- **Warfarin INR**: 2–3 (most) · 2.5–3.5 (mechanical mitral valve)
- **IV K⁺**: typical peripheral max ~10 mEq/hr; **NEVER IV push** — fatal
- **Insulin + glucose for hyperK**: standard order — shifts K⁺ into cells; watch glucose

## 🚨 Sepsis Hour-1 bundle (memorize)

**L · C · A · F · V**
**L**actate · **C**ultures (before abx) · **A**ntibiotics (early!) · **F**luids 30 mL/kg crystalloid · **V**asopressors if MAP <65

**SIRS** = 2+ of: T <36 or >38 · HR >90 · RR >20 (or PaCO₂ <32) · WBC <4 or >12 (or >10% bands)

## 🫀 CPR / ACLS quick

- **Compression depth**: 2–2.4 in (5–6 cm) · **rate** 100–120/min · **C:V ratio** 30:2 (no advanced airway)
- **Defib**: pulseless VT / VF · **Cardiovert (synced)**: unstable tachy *with a pulse*
- **Atropine** for symptomatic brady · **Epi** in arrest (1 mg q3–5 min)

## ⚡ "Call now" critical-value cheat list

K⁺ <2.5 or >6.0 · Na⁺ <125 or >155 · pH <7.30 or >7.50 · Glucose <70 or >250
HR <50 or >120 sustained · SBP <90 or MAP <65 · SpO₂ <90% (<88 in COPD)
Lactate ≥4 · Troponin trending up · GCS drop ≥2 · Urine <0.5 mL/kg/hr × 2 h
