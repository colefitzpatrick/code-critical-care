# 01 · The Big Picture

## One-sentence model
> **The brain is a high-demand organ in a rigid box: it needs constant perfusion, and any
> extra volume inside the skull steals that perfusion by raising pressure.**

## The two equations + one doctrine (the whole module)
- **CPP = MAP − ICP** — cerebral perfusion pressure. Keep ≥ ~60. *(Net pressure driving brain blood flow.)*
- **Monro-Kellie doctrine** — the skull is a fixed box holding 3 things: **brain + blood + CSF**. Add volume to one (tumor, bleed, edema) → the others must shrink, then **ICP spikes**.
- Normal **ICP 7–15 mmHg**; treat **>20**.

## Engineering translation
| Physiology | Your language |
|---|---|
| CPP = MAP − ICP | Net driving pressure = inflow − back-pressure |
| Monro-Kellie | Sealed rigid pressure vessel — small added volume → nonlinear pressure rise |
| CO₂ = cerebral vasodilator | ↑CO₂ → ↑cerebral blood volume → ↑ICP (so avoid hypercapnia) |
| Brain has no glycogen/O₂ store | Zero buffer capacity → fails in seconds without flow |
| Autoregulation | Brain holds steady flow across a MAP range — until injury breaks it |

## The spine
```
Brain needs constant flow (no fuel reserve)
        ↓ flow = CPP = MAP − ICP
Anything ↑ ICP (bleed, mass, edema, ↑CO₂, low Na⁺) → ↓ CPP → ischemia
        ↓ the body's last-ditch reflex
Cushing's triad (↑SBP/wide pulse pressure, bradycardia, irregular breathing) = LATE = herniation near
        ↓ break it down by problem
Stroke (flow blocked or bleeding) · Seizure (electrical storm) · ↑ICP (pressure)
        ↓
each has a predictable assessment + a time-critical nurse action
```

## 5 anchors
1. **CPP = MAP − ICP.** Raise MAP or lower ICP to perfuse the brain.
2. **Monro-Kellie:** fixed box — extra volume → pressure → herniation.
3. **CO₂ dilates cerebral vessels** → never let an at-risk brain hypoventilate.
4. **Stroke is "time = brain"**: ischemic (clot, give tPA in window) vs hemorrhagic (bleed, NO tPA).
5. **GCS ≤ 8 → intubate** (can't protect airway). (Reference doc has the scale.)
